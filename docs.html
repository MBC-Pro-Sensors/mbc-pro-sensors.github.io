01. 一鍵智能校準 (SOP)

為確保高解析度數值的準確性，每次更換場地或開機時，請務必執行校準：

  • 步驟一： 將感應器安裝至機器人上，並確保感應器底部距離地面約 0.8 公分。
  • 步驟二： 按住感應器上的實體「校準按鈕」。
  • 步驟三： 將機器人平移，讓感應器完整掃過場地上的「最黑的線」與「最白的底色」區域。
  • 步驟四： 掃描完成後，放開按鈕。系統會自動將極值寫入 EEPROM，校準即刻生效。

02. PyBricks Python 開發指南

匯入我們提供的 lineSensorPyFunc.py 後，您可以在 SPIKE / EV3 的 Python 環境中輕鬆呼叫高階函式：

from pybricks.iodevices import PUPDevice
from pybricks.parameters import Port
from lineSensorPyFunc import getLineBlackError, getLineSensor5

# 初始化感應器 (連接至 Port C)
lineSensorDevice = PUPDevice(Port.C)

while True:
    # 1. 取得高解析度位置誤差 (範圍: -100 到 100)
    # 完美適用於 PID 循線演算法的 Error 值
    error = getLineBlackError()
    
    # 2. 讀取單顆感應器的原始反射值 (索引 1~5，數值 0~100)
    sensor_left = getLineSensor5(1)
    sensor_right = getLineSensor5(5)
    
    print("誤差值:", error, " | 最左側數值:", sensor_left)

03. 核心數據定義

  • getLineBlackError()：回傳黑線的中心位置誤差。數值若為 0 表示正中心，正負值對應偏移方向（依您設定的軟硬體方向而定，通常為 ±50 或 ±100）。
  • getLineBlackWidth()：回傳當前偵測到的黑線寬度，範圍 0~5。
  • getLineSensor5(index)：讀取指定感應器（1左至5右）的紅外線反射量，黑線通常接近 0，白底接近 100。
