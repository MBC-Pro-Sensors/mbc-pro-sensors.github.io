HARDWARE CALIBRATION (一鍵智能校準)

為確保高解析度數值的準確性，每次更換場地材質或現場光源改變時，請務必執行校準程序：

  • 步驟一 [高度定位]： 將感應器安裝至機器人底盤，並確保感應器底部距離場地地面約 0.8 公分至 1.2 公分 之間。
  • 步驟二 [啟動寫入]： 按住感應器上的實體「校準按鈕」不放，此時狀態指示燈將閃爍。
  • 步驟三 [橫向掃描]： 保持按住按鈕，將機器人平移，讓感應器完整掃過場地上的「最黑的線」與「最白的底色」區域。
  • 步驟四 [校準完成]： 掃描完成後，放開按鈕。系統會自動將極值寫入 EEPROM 晶片，校準即刻生效，斷電不遺失。
📥 下載 SPIKE 官方範例檔 (.llsp3)

我們提供完整的 PyBricks Python 函式庫，讓您能輕鬆讀取高解析度誤差值。

Python 快速上手 (PyBricks):

from pybricks.iodevices import PUPDevice
from pybricks.parameters import Port
from lineSensorPyFunc import getLineBlackError, getLineSensor5

# 初始化感應器 (連接至 SPIKE Port C)
lineSensorDevice = PUPDevice(Port.C)

while True:
    # 1. 取得高解析度位置誤差 (範圍: -100 到 100)
    # 非常適合直接套入 PID 公式：TurnRate = Error * Kp
    error = getLineBlackError()
    
    # 2. 讀取最外側兩顆感應器的原始反射值 (索引 1 與 5)
    sensor_left = getLineSensor5(1)
    sensor_right = getLineSensor5(5)
    
    print("誤差值:", error, " | L:", sensor_left, " R:", sensor_right)

📥 下載 EV3-G 匯入積木檔 (.ev3b)

透過 I2C 或 UART 轉接板，五路感應器可完美相容於 LEGO EV3 系統。

  • 連線方式： 使用標準 EV3 傳輸線連接至 Input Port 1~4。
  • 通訊協定： 預設採用 UART 模式通訊，提供極低延遲的數據傳輸。
  • 積木匯入： 下載上方 `.ev3b` 檔案，透過 EV3 軟體的「工具 -> 區塊匯入嚮導」進行安裝。安裝後即可在感應器面版中找到「MBC 5-Way Line Sensor」。

專為 Makeblock 體系打造的極致相容性，讓 mBot / mBot Neo 也能擁有競賽級的循線能力。

  • 連線方式： 透過 RJ25 轉接模組，連接至 mBot 主板的任意擴充埠 (Port 1 ~ 4)。
  • 擴充安裝： 開啟 mBlock 5 軟體，點擊「擴充」，搜尋「MBC SENSOR」，點擊添加即可獲得專屬積木。
