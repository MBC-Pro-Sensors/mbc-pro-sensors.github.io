Welcome to the EV3 zone! The Pathfinder 8-Way Sensor perfectly supports the following mainstream EV3 software environments. Please click on the environment that matches your setup to view the corresponding installation and block programming tutorials:

> [!NOTE]
> In the EV3 system, the bottom layer of this sensor **fully emulates the original LEGO EV3 Infrared Sensor (IR Sensor)**. This means you do not need to install any third-party blocks; you can directly use the built-in infrared blocks to read all the data!
