> **告別超音波鬼影：180度零死角，相撲對戰與極速避障的雷射雷達。**

## 🚀 產品概述

超音波感應器在機器人競賽中有兩個致命缺點：**反應慢**（音速限制）與**容易產生回波鬼影**（干擾數據）。在相撲對戰或高速避障的場景中，這兩個缺點可能直接導致比賽失敗。

測距者 8 路搭載 8 顆雷射測距單元，涵蓋**完整的左右各 90 度，共 180 度的全視角**，讓機器人對環境擁有接近全知的空間感知能力，是目前市場上最完整的樂高相容雷射測距解決方案。

## 🎯 核心優勢

- 🌐 **180 度零死角覆蓋**：左右各 4 顆，涵蓋完整的前方半圓範圍，無死角偵測所有威脅
- ⚡ **雷射級極速反應**：以光速測距，毫秒級反應，讓機器人在最快的速度下做出判斷
- 🥊 **相撲對戰最強武器**：全視角讓對手無所遁形，第一時間鎖定目標並追擊
- 🔼 **無縫升級設計**：具備雙 0 度（正前方）雷射點，與 TOF2 程式邏輯完美銜接，升級零成本

## 📐 角度分佈說明

8 顆雷射測距單元以固定角度分佈，涵蓋完整的 180 度視角：

| 通道 | 角度 | 方向 |
| :---: | :---: | :--- |
| 1 | -90° | 正左側 |
| 2 | -60° | 左前 60° |
| 3 | -30° | 左前 30° |
| 4 | 0° | 正前方 |
| 5 | 0° | 正前方（與 TOF2 銜接用） |
| 6 | +30° | 右前 30° |
| 7 | +60° | 右前 60° |
| 8 | +90° | 正右側 |

> **💡 設計巧思**：保留兩顆正前方（0 度）感應器，是為了讓從 TOF2 升級的使用者可以無縫延續原本的程式邏輯，只需新增額外通道的處理即可。

## 🔌 硬體接線

1. 使用標準連接線插上 SPIKE（Port A~F）或 EV3（Port 1~4）
2. 通電後確認感應器指示燈正常亮起即完成

## ❓ 常見問題 FAQ

- **❓ 讀到奇怪的超大數值？** → 表示該方向沒有偵測到物體，屬於正常現象。可在程式中設定最大閾值過濾。
- **❓ 同一角度兩個感應器數值不同？** → 兩顆 0 度感應器的細微角度差異可能造成數值略有不同，建議取平均值使用。
