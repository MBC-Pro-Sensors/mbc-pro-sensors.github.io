> **突破主機孔位極限：自帶隔離電源保護，專為 Python 進階專案打造的 6 路擴充神兵。**

## 🚀 產品概述

SPIKE Prime 主機只有 6 個連接孔，但大型機器人專案往往同時需要多顆感應器加上多個馬達，孔位根本不夠用。

SPIKE 6 路多工器讓您用一個孔位同時連接多達 6 組感應器或馬達。更重要的是，本產品搭載**獨立供電系統**，馬達所需的大電流不會從主機抽取，**完全不會因為馬達耗電導致主機當機**，同時內建**隔離保護電路**，確保任何接線問題都不會損傷您昂貴的 SPIKE 主機。

## 🎯 核心優勢

- 🔌 **突破孔位限制**：1 個主機 Port 擴充為 6 個可用接口
- ⚡ **獨立隔離電源**：馬達電流由獨立電源供應，不從主機抽電，徹底消除當機風險
- 🛡️ **隔離保護電路**：電氣隔離設計，保護 SPIKE 主機不受異常電流損害
- 🐍 **Python 深度整合**：支援 Pybricks Python 環境，提供完整的多工器控制能力

## ⚠️ 重要限制說明

> [!WARNING]
> **請在購買前確認您的使用場景符合以下規格：**

| 項目 | 說明 |
| :--- | :--- |
| **相容主機** | SPIKE Prime 限定（不支援 Robot Inventor 與 EV3） |
| **SPIKE 官方軟體** | 僅支援 **Python 文字模式**，不支援圖控積木模式 |
| **Pybricks 環境** | 僅支援進階 Python 讀取，不支援簡單圖形指令 |

> 💡 **適合對象**：已熟悉 Python 程式設計、需要大量感應器與馬達的進階開發者與競賽選手。若您主要使用圖控積木，本產品目前**不適合**您的需求。

## 🔌 硬體接線

1. 將多工器插入 SPIKE 主機的任一 Port
2. 接上獨立電源（含電池盒或變壓器）
3. 將您的感應器與馬達接入多工器的 6 個擴充口
4. 在 Python 環境中使用對應指令讀取各口數據
